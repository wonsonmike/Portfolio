About This Project

This is the personal portfolio website I built to showcase my programming skills and projects to potential employers. It reflects my ability to build clean, responsive, and modern websites using core web technologies.

Key Features

  • Custom multi-page site with a minimalist, professional design.
  • Dark and light theme toggle with user preference saved across pages.
  • Sticky navigation bar that stays visible while scrolling for easy navigation.
  • Separate detailed pages for each project to highlight my work clearly.
  • Fully responsive and hosted for free using GitHub Pages.

Technologies Used

  • JavaScript
