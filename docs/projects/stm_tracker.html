STM Tracker

About This Project

While there are many basic period tracking apps, my plan is to use a more advanced STM (Sympto-Thermal Method) tracking, which tracks more than just the number of days to the next period, using a variety of indicators of the point at which a woman is in their cycle. The main features will be a daily input for the user, calculation of the point in the cycle and fertility likelihood, and cycle history.

The existing solution to STM tracking is by manually tracking various indicators, such as internal temperature and mucus features, creating a graph and understanding what step in the cycle they’re in based on these manually tracked details. My solution would take the work off of the individual and automate it. This way, no work other than daily data entry is required to know all details of their cycle in accordance with the STM. This would have a real-world impact of making the more accurate STM tracking much more accessible to people.

The intended audience/customer/user of this app is women in general, with the primary audience being women that want to use natural family planning (NFP) methods, such as STM. This more accurate fertility tracking method can be used by all women in need of accurate menstrual tracking.

Key Features

  • React Native offers cross-compatibility with Android and iOS
  • Simple, straightforward data entry for the user
  • Automated cycle tracking using the complex STM tracking
  • Robust error handling and algorithm
